# Chatting App

This project is a simple full-stack chat application that supports user registration, login, normal messaging, and an AI assistant that replies to the user.  
The goal is to keep everything clean, understandable, and easy to extend.

---

## Project Overview

The application has two main parts:

### 1. Client Side (HTML, CSS, JavaScript)
- Login / Register pages
- Chat page
- Axios for API calls
- Simple responsive UI
- JavaScript functions to send and display messages

### 2. Server Side (PHP + MySQL)
- Authentication system
- Conversations and messages logic
- AI response system
- MVC-style organization
- Migrations to generate DB tables
- Controllers, Models, Services, Middleware

---

## Features

- Register new users  
- Login with validation  
- Send messages  
- Messages appear instantly  
- AI assistant reply  
- Protected routes using middleware  
- Organized folder structure  
- Axios for all API calls

---

## Installation Guide

### 1. Move the project
Place the project folder into your server directory:
- XAMPP: `htdocs`

### 2. Create the database
Create a database named: "chatting_app"
Run the migration files manually, or insert your own SQL dump.

open this is your browser:
http://localhost/chatting_app/client-side/pages/login.html
